# 📱 Web-basierte QR/Barcode ScannerApp

## 🎯 Projektbeschreibung

Eine einfache, browserbasierte Webanwendung zum Scannen von QR-Codes und Barcodes. Der gescannte Inhalt wird automatisch an eine Ziel-URL übergeben und das Ergebnis im Browser angezeigt.

**Technologie-Stack:**
- HTML5 + CSS3 + JavaScript
- ZXing Library (@zxing/library)
- LAMP-Server (Linux, Apache, MySQL, PHP)
- Mobile-first Design

---

## 📋 Voraussetzungen

- Webserver mit HTTPS (Let's Encrypt empfohlen)
- Moderne Browser (Chrome, Safari, Firefox)
- Mobile Devices: iOS 11+ / Android 5.0+
